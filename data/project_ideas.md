# Project Ideas

## Local RAG Chat Assistant
Build a local retrieval-augmented generation (RAG) chatbot that answers questions from your own documents. Use Python, LangChain, Ollama, and Chroma. Support multiple document types and show which files were used to answer each question.

## Coding Interview Practice Tracker
Create a web app that tracks coding interview problems solved across platforms such as LeetCode and HackerRank. Store metadata such as topic, difficulty, time taken, and mistakes. Provide charts showing progress over time and weak areas.

## System Design Notes Explorer
Store system design notes, diagrams, and examples in a searchable format. Provide a small web interface where the user can search by concept, such as caching, load balancing, or database sharding, and see related notes and diagrams.

## Personal Knowledge Base with Tags
Implement a personal knowledge base where each note can be tagged with topics. Allow full-text search and filtering by tags. Optionally, add embeddings and semantic search so similar notes are suggested automatically.
